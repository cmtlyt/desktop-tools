乐清定制麻将 · 最终完整定稿规则（亲友专用版）
 
一、基础设定
 
1. 人数：4人，使用136张标准麻将（筒、条、万、东南西北中发白）
2. 唯一合法胡型：n组面子（顺/刻/碰/杠） + 一对将
3. 七对禁止胡牌
4. 吃碰权限：只能吃上家；碰牌不限任意玩家
 
二、花牌规则
 
1. 红中、白板为固定花牌，摸到必须立刻杠牌补牌
2. 若红中/白板其中一张为本局财神，则由发财替代充当花牌，同样必须强制杠
 
三、财神规则（终极定稿）
 
1. 全员摸完手牌后，从牌山最后末尾取一张为本局财神
2. 财神取出后不回归牌山，本局场上最多流通3张财神
3. 财神归位仅全顺子牌型生效，带刻子、碰牌不判定归位
4. 财神归位只加台数、不算硬牌，不改变手牌软硬属性
5. 核心铁规：手牌只要带财神，一律为软牌
 
四、开局掷骰 & 摸牌规则
 
1. 庄家先掷一次骰子，骰子指向的玩家再掷第二次
2. 两次点数相加，从第一次骰子指向玩家的牌山左侧向右数对应对数
3. 庄家从下一对开始摸牌，全程每次摸2对（4张）
4. 四家各摸满8对（16张），庄家额外多摸1张：庄家17张、闲家16张
5. 所有杠牌、花牌补杠，统一从牌山末尾补牌
 
五、吃、碰、杠 & 特殊胡牌
 
1. 吃牌：仅限上家打出的牌
2. 碰牌：所有玩家打出均可碰
3. 普通杠不计台数
4. 支持：抢杠胡、杠上开花（均单独加台）
 
六、流局规则
 
牌山剩余8对牌时，直接荒庄流局
 
七、庄家 & 连庄规则
 
1. 庄家输牌，下家接任庄家
2. 连庄无次数上限，连庄台数持续累积叠加，底分不翻倍
3. 结算无自摸、点炮区别：庄家对三家闲家台数统一一致
 
八、软硬牌判定（最新终极修订）
 
硬牌（基础1台）
 
满足手牌无财神，即为硬牌：
 
1. 普通无财神胡牌
2. 无财神碰碰胡、无财神混一色、无财神清一色（三类牌型本身均为硬牌基数）
3. 财神归位不纳入硬牌判定
 
软牌（基础0.5台）
 
1. 只要手牌带有财神，全部判定为软牌
2. 带财神的碰碰胡/混一色/清一色，统一变软牌，且比无财神硬牌同型少1台
 
九、基础牌型台数（硬牌基准台数）
 
1. 基础硬牌：1台
2. 基础软牌：0.5台
3. 碰碰胡：有财神3台 / 无财神4台（硬牌基准）
4. 混一色：3台（硬牌基准）
5. 清一色：4台（硬牌基准）
 
十、通用固定加台（全部可累加）
 
1. 单张花牌：+1台/张，多张累加
2. 杠上开花：+1台
3. 抢杠胡：+1台
4. 自家风位 碰/刻/杠：+1台
5. 发财 碰/刻/杠：+1台
 
十一、财神专属叠加台数
 
1. 双财神：+1台
2. 财神归位：+1台（仅限全顺子、不算硬牌）
3. 双财神归位：+6台
4. 三财神归位：+50台
5. 双财神做将：+6台
6. 三财神做刻：+15台
 
十二、特殊牌型预留项（待补台数）
 
四张相同花、八张全花、天胡、地胡、海底捞月、边张顺子
 
十三、终极结算规则（独家定制）
 
1. 小数补整：0.5台直接进1补整为1台
2. 庄家加成：庄家胡牌，总台数额外+1台
3. 闲家胡牌规则：闲家胡牌自身台数+1台；仅与庄家抵扣台数差，其余闲家正常输庄家、输胡牌闲家
 
示例：庄家累积8台，闲家胡4台 → 闲家结算5台，仅输庄家3台；其余闲家输庄家8台、输胡牌闲家4台
 
4. 高台翻倍：总台数≥6台，整体台数直接翻倍
 
示例：总7台 → 结算14台
